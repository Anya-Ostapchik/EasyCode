Краткое описание

Продолжительность курса —48 занятий (12 месяцев)


Курс "Разработка мобильных приложений" предназначен для тех, кто хочет освоить создание современных приложений для iOS и Android. Вы изучите ключевые технологии и фреймворки, такие какSwift,Kotlin,Flutter иReact Native.


На курсе вы познакомитесь с основными этапами разработки мобильных приложений:

  • Проектирование интерфейсов — создание удобных и современных UI/UX-дизайнов.
  • Программирование — разработка функционала приложений на Swift (для iOS) и Kotlin (для Android).
  • Кроссплатформенная разработка — использование Flutter и React Native для создания приложений, работающих на обеих платформах.
  • Публикация — подготовка и размещение приложений в App Store и Google Play.

По итогу 48 занятий высоздадите 4-5 собственных проектов, которые станут частью вашего портфолио. Вы научитесь разрабатывать приложения с нуля, тестировать их и публиковать в магазинах приложений.


Навыки, которые вы приобретете:
  • умение разрабатывать приложения для iOS и Android;
  • навыки работы с современными фреймворками (Flutter, React Native);
  • понимание принципов кроссплатформенной разработки;
  • умение проектировать интерфейсы и создавать удобный UX;
  • навыки публикации приложений в App Store и Google Play.

Разработка под iOS (16 уроков)

Цели курса:
  • изучение языка Swift и его основных конструкций;
  • освоение среды разработки Xcode;
  • создание интерфейсов с использованием SwiftUI и UIKit;
  • работа с базами данных (CoreData, Realm);
  • интеграция API и работа с сетью.

Профессиональные навыки, которые вы приобретете:
  • умение разрабатывать приложения для iOS;
  • навыки работы с Swift и Xcode;
  • понимание принципов создания интерфейсов;
  • умение интегрировать API и работать с данными.

Разработка под Android (16 уроков)

Цели курса:
  • изучение языка Kotlin и его основных конструкций;
  • освоение среды разработки Android Studio;
  • создание интерфейсов с использованием Jetpack Compose и XML;
  • работа с базами данных (Room, SQLite);
  • интеграция API и работа с сетью.

Профессиональные навыки, которые вы приобретете:
  • умение разрабатывать приложения для Android;
  • навыки работы с Kotlin и Android Studio;
  • понимание принципов создания интерфейсов;
  • умение интегрировать API и работать с данными.

Кроссплатформенная разработка (16 уроков)

Цели курса:
  • изучение фреймворка Flutter и языка Dart;
  • освоение React Native и JavaScript для кроссплатформенной разработки;
  • создание приложений, работающих на iOS и Android;
  • интеграция API и работа с состоянием приложения.

Профессиональные навыки, которые вы приобретете:
  • умение разрабатывать кроссплатформенные приложения;
  • навыки работы с Flutter и React Native;
  • понимание принципов управления состоянием приложения;
  • умение интегрировать API и работать с данными.
