Краткое описание

Продолжительность курса — 48 занятий (12 месяцев)


Курс "Автоматизация и DevOps" предназначен для тех, кто хочет освоить современные подходы к автоматизации процессов разработки и эксплуатации приложений. Вы изучите ключевые инструменты и технологии, такие как Docker, Kubernetes, Terraform, Ansible, Jenkins и GitLab CI/CD.


На курсе вы познакомитесь с основными этапами DevOps:

  • Автоматизация сборки и развертывания — настройка CI/CD для ускорения процессов разработки.
  • Управление инфраструктурой — работа с облачными платформами и инструментами Infrastructure as Code (IaC).
  • Контейнеризация — использование Docker и Kubernetes для управления приложениями.
  • Мониторинг и логирование — настройка систем для отслеживания производительности и устранения неполадок.

По итогу 48 занятий вы создадите 4-5 собственных проектов, которые станут частью вашего портфолио. Вы научитесь автоматизировать процессы, управлять инфраструктурой и внедрять DevOps-практики в реальные проекты.


Навыки, которые вы приобретете:
  • умение настраивать и использовать CI/CD-пайплайны;
  • навыки работы с контейнерами и оркестрацией (Docker, Kubernetes);
  • понимание принципов Infrastructure as Code (Terraform, Ansible);
  • умение настраивать мониторинг и логирование;
  • навыки работы с облачными платформами (AWS, Azure, GCP).

Основы автоматизации (16 уроков)

Цели курса:
  • изучение основ CI/CD и их роли в DevOps;
  • освоение инструментов для автоматизации сборки и развертывания (Jenkins, GitLab CI/CD);
  • работа с системами контроля версий (Git);
  • настройка автоматических тестов и деплоя.

Профессиональные навыки, которые вы приобретете:
  • умение настраивать CI/CD-пайплайны;
  • навыки работы с Git и системами контроля версий;
  • понимание процессов автоматизации сборки и развертывания;
  • умение интегрировать автоматические тесты в CI/CD.

Контейнеризация и оркестрация (16 уроков)

Цели курса:
  • изучение основ контейнеризации с использованием Docker;
  • освоение оркестрации контейнеров с Kubernetes;
  • работа с Docker Compose для управления многоконтейнерными приложениями;
  • настройка кластеров Kubernetes и управление ими.

Профессиональные навыки, которые вы приобретете:
  • умение создавать и управлять контейнерами;
  • навыки работы с Kubernetes для оркестрации;
  • понимание принципов работы кластеров и подов;
  • умение масштабировать приложения в Kubernetes.

Infrastructure as Code и облачные технологии (16 уроков)

Цели курса:
  • изучение принципов Infrastructure as Code (IaC);
  • освоение Terraform для управления инфраструктурой;
  • работа с Ansible для автоматизации конфигураций;
  • интеграция с облачными платформами (AWS, Azure, GCP).

Профессиональные навыки, которые вы приобретете:
  • умение управлять инфраструктурой с использованием Terraform;
  • навыки автоматизации конфигураций с Ansible;
  • понимание работы облачных платформ и их интеграции;
  • умение создавать и управлять инфраструктурой в облаке.
