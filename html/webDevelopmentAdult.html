Краткое описание

Продолжительность курса — 48 занятий (12 месяцев)


Все сайты и веб-приложения состоят из двух частей: frontend и backend. На нашем курсе вы изучите frontend, включая языки разметки HTML и CSS, которые отвечают за структуру и оформление элементов, например, расположение и стиль заголовков.


Также курс включает изучение языка программирования JavaScript, который добавляет интерактивность сайту — кнопки, галереи, анимации и многое другое.


Любой сайт или веб-приложение создается с использованием этих трех ключевых элементов веб-разработки.

По итогу 48 занятий вы создадите 4-5 собственных проектов. Вы изучите все ключевые этапы разработки: дизайн, верстку, программирование и публикацию на хостинг. Вы научитесь создавать интернет-магазины, блоги, лендинги и многостраничные сайты, а также поймете, какой тип сайта лучше подходит для конкретных задач.


Навыки, которые вы приобретете:
  • умение четко формулировать задачи и цели;
  • навыки работы в команде и управления проектами;
  • умение анализировать и решать проблемы;
  • развитие критического мышления;
  • улучшение навыков презентации и защиты проектов.

HTML + CSS (32 урока)

Цели курса:
  • изучение основных структурных элементов сайта;
  • освоение работы с кодом в редакторе VS Code;
  • практическое применение тегов HTML;
  • создание и подключение CSS-файлов;
  • верстка сайта блоками и их стилизация с помощью CSS;
  • освоение инструментов прототипирования и дизайна;
  • верстка сайта по макету с переносом свойств в CSS;
  • размещение проектов на бесплатном хостинге.

Профессиональные навыки, которые вы приобретете:
  • понимание этапов создания сайтов;
  • знание основных тегов HTML и принципов блочной верстки;
  • умение работать с CSS для оформления элементов;
  • навыки самостоятельной работы над проектами.

JavaScript (16 уроков)

Цели курса:
  • изучение фреймворков и библиотек для упрощения разработки;
  • освоение основ программирования и алгоритмов;
  • изучение синтаксиса и основных конструкций JavaScript;
  • применение JavaScript для создания интерактивных элементов сайта;
  • разработка полноценных страниц с использованием JavaScript;
  • создание игровых механик на JavaScript.

Профессиональные навыки, которые вы приобретете:
  • написание JavaScript-кода для решения задач;
  • анализ и оптимизация кода;
  • работа с библиотеками и фреймворками;
  • создание анимаций и обработка событий на JavaScript.

Фреймворки: React или Vue.js (12 уроков)

Цели курса:
  • освоение одного из популярных фреймворков для frontend-разработки;
  • изучение компонентного подхода к созданию интерфейсов;
  • практическое применение фреймворка для создания современных веб-приложений;
  • интеграция фреймворка с существующими проектами.

Описание фреймворков:
  • React — библиотека для создания пользовательских интерфейсов, разработанная Facebook. React использует компонентный подход, что позволяет создавать масштабируемые и легко поддерживаемые приложения. Он идеально подходит для разработки сложных SPA (Single Page Applications) и активно используется в крупных проектах.
  • Vue.js — прогрессивный фреймворк для создания интерфейсов, который отличается простотой интеграции и гибкостью. Vue.js подходит как для небольших проектов, так и для крупных приложений. Он имеет интуитивно понятный синтаксис и активно развивается благодаря поддержке сообщества.

Профессиональные навыки, которые вы приобретете:
  • умение создавать компоненты и управлять их состоянием;
  • работа с инструментами сборки и управления зависимостями;
  • интеграция API и работа с данными в реальном времени;
  • оптимизация производительности приложений.
